Write a function to find the longest common prefix string amongst an array of strings.

If there is no common prefix, return an empty string "".
NOTE: prefix means the starting portion of the string.

 

Example 1:
Input: strs = ["flower","flow","flight"]
Output: "fl"

Example 2:
Input: strs = ["dog","racecar","car"]
Output: ""
Explanation: There is no common prefix among the input strings.

Example 3:
Input: strs = ['car', 'my car', 'his car']
Output: ""
Explanation: There is no common prefix among the input strings. the common parts are at the end of the strings not the beginning.
